Code

Drogulus is currently an unfinished work in progress.

Source code is hosted on Github. Patches, pull requests, bug reports, comments, suggestions and good ideas most welcome!

Version Log

  • 0.0.0.alpha.0 (19th August 2011): The release number used during planning and initial implementation.
    • Implementation of Kademlia based distributed hash table.
    • Lots of documentation.
    • Implementation of crypto based trust mechanism.
    • A simple shim of a Lisp written in Python created as a proof-of-concept rather than sensible solution.
